# 💈 BarberGo - Admin Module

**BarberGo** adalah aplikasi mobile Flutter untuk mengelola operasional barbershop. Modul ini khusus untuk **Admin**, memungkinkan pengelolaan data pelanggan, pemesanan, pembayaran, layanan, portofolio, dan jadwal barber.

## 🔧 Teknologi yang Digunakan
- **Flutter** + **Bloc Pattern**
- **REST API Integration**
- **Clean Architecture** (Presentation, Domain, Data)
- **Google Fonts**, **Custom Icons**
- **Multi-role (Admin, Pelanggan, Barber)**
- Locale Indonesia (`id_ID`) untuk tanggal & mata uang
